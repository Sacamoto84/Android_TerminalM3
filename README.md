# TerminalM3

`TerminalM3` - Android-приложение для приема и отображения текстовых данных от сетевых устройств в формате терминала. Проект подходит для отладки ESP, контроллеров и другой электроники, когда нужно видеть поток сообщений в реальном времени, разбирать ANSI-последовательности и быстро открывать встроенный веб-портал устройства.

## Возможности

- Живой консольный экран с автоматической прокруткой к последним строкам.
- Отображение ANSI / ESC-последовательностей: 256 цветов, цвет фона, `bold`, `italic`, `underline`, мигание.
- Показ номеров строк и символов `CR` / `LF` для отладки входящего потока.
- Быстрая очистка консоли и переход к концу лога.
- Отображение IP-адреса телефона и адреса устройства.
- Открытие веб-портала устройства прямо внутри приложения.
- Экран-справка с палитрой цветов и примерами escape-кодов.
- Работа с сетевыми сообщениями по UDP.

## Для чего нужен проект

Приложение можно использовать как компактный мобильный терминал для устройств, которые отправляют текстовые статусы, сервисные сообщения и цветные логи по сети. Это удобно, когда нужно быстро подключиться к устройству, посмотреть поток данных, проверить форматирование строк и при необходимости перейти в его web-интерфейс.

## Сборка и запуск

Требования:

- Android Studio
- JDK 17
- Android 6.0+ (`minSdk 23`)

Как запустить:

1. Клонировать репозиторий.
2. Открыть проект в Android Studio.
3. Дождаться синхронизации Gradle.
4. Запустить конфигурацию `app` на устройстве или эмуляторе.

## Особенности

- Имя приложения: `TerminalM3`
- Версия проекта: `3.0.0`
- Основной сетевой сценарий в текущей реализации использует UDP-порт `8888`
